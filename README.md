# Restful-API
 Testing restful API on the TypeScript.
 
## Technology stack
- Express.
- TypeScript (tsc).
- Body-Parser.
- Dotenv.
- Nodemon.

## Project setup
```
npm install
```

### Launching a project in developer mode
```
npm run dev
```

### Build a project
```
npm run build
```

### Run built project
```
npm run start
```

#### Create a standard tsconfig.json
```
npx tsc --init
```
